Pure CSS3 Round Menu

Overview and features

249 vector icons. It means that Menu is retina ready and looks perfect on any zoom. You can find all available icons here.

Tested and works in all modern browsers and platforms: Mac OS (sifari, chrome, firefox, opera), Windows OS (sifari, chrome, firefox, opera, IE8, IE9, IE10), iPhone, iPad, Android.

5 practical exaples that can be used as base for your own menu.

Unlimited color combinations and 12 predefined colors.

16 beautiful animation effects with pure CSS3.

9 position options with different number of maximum elements.

Tooltips with 6 available positions.

Round, ellipse, rounded square and square shapes for menu buttons.
